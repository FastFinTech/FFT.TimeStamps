Struct TimeOfWeek

Add(TimeSpan)

Adds the given amount of time to the TimeOfWeek and returns the new TimeOfWeek. The result wraps around the end of the week back to the beginning, so the result of an add operation could be "less than" the original value. You can add positive or negative time values. Negative time values will also "wrap backwards" around the week. The result will always be less than EndOfWeek.
